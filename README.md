# BidSphere - Online Auction System

Project Overview:

BidSphere is a full-stack online auction platform that enables users to create auctions, place bids, and participate in real-time bidding activities through a modern and responsive web interface.

The platform provides a secure environment for buyers and sellers while ensuring a smooth auction experience.

Features:

* User Registration and Login Authentication
* Role-Based Access Control
* Create and Manage Auctions
* Real-Time Bidding System
* Product Image Upload
* Auction Status Tracking
* User Dashboard
* Responsive User Interface
* Secure Backend APIs
* MongoDB Database Integration

Tech Stack:

### Frontend

* React.js
* Tailwind CSS
* JavaScript
* Axios

Backend:

* Node.js
* Express.js

Database:

* MongoDB

Other Tools:

* Cloudinary (Image Storage)
* Git & GitHub

## Installation

### Clone Repository

git clone https://github.com/anshitachaudhary296-svg/BidSphere-Online-Auction-System.git

### Install Dependencies

Frontend:

cd frontend
npm install

Backend:

cd backend
npm install

### Run Project

Frontend:

npm run dev

Backend:

npm start
